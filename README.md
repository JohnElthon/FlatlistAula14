# FlatlistAula14

Projeto de estudo sobre uso do componente FlatList em React Native.

## Sobre o projeto

O **FlatlistAula14** foi desenvolvido durante meus estudos em React Native, com o objetivo de praticar conceitos vistos em aula e melhorar minha lógica no desenvolvimento de aplicativos mobile.

Este projeto faz parte da minha evolução como estudante de Desenvolvimento de Sistemas.

## Funcionalidades

- Renderização de listas
- Uso de FlatList
- Organização de dados
- Prática de componentes

## Tecnologias utilizadas

- React Native
- JavaScript
- Expo
- Node.js / npm

## Como executar o projeto

1. Clone este repositório:

```bash
git clone https://github.com/JohnElthon/FlatlistAula14.git
```

2. Acesse a pasta do projeto:

```bash
cd FlatlistAula14
```

3. Instale as dependências:

```bash
npm install
```

4. Execute o projeto:

```bash
npm start
```

5. Abra no emulador Android/iOS ou no aplicativo Expo Go.

## Observação

Projeto desenvolvido com fins acadêmicos e de aprendizado. Algumas partes podem representar práticas simples de aula, com foco em entender conceitos específicos do React Native.
